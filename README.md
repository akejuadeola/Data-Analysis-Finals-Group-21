# Data-Analysis-Finals-Group-21
**Final Class Project**

*Repository Contents*
1. **Python Files** - The repository contains two Python scripts used for data preparation, analysis, and visualization.
2. **Input Dataset** - happiness_2024.csv is the primary dataset required to run the analysis.
3. **GDP Dataset** - worldbank_gdp.csv is included for reference; however, the Python code can regenerate this dataset automatically if needed.
4. **Main Project File** - happiness_project_group21.py contains the complete final project workflow and analysis.
5. **SQL Queries** - All standalone SQL queries used in the project are included in sql_queries_group21.py.
6. **PDF Presentation** - Final presentation of analysis to be discussed in class

*Project Overview*

This project analyzes global happiness indicators and their relationship with economic factors such as GDP per capita. The analysis was conducted using Python, SQL, and data visualization techniques.
